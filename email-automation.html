🔑 Resend Sender Settings
ℹ️
Setup: the Resend API key lives on the server, not here — browsers can't call Resend's API directly (it has no CORS support, so pasting a key into a page field would just fail with "Failed to fetch", and would also expose the secret in your browser).

In your Vercel project → Settings → Environment Variables, set RESEND_API_KEY to your key from resend.com → API Keys, then redeploy. Verify your sending domain under Resend → Domains first.
