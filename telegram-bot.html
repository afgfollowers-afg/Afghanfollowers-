How to Set Up
1
Create a Bot
Open Telegram → search @BotFather → send /newbot → follow instructions → copy the Bot Token
2
Get Your Chat ID
Send /start to your bot → come back here → click "Get My Chat ID" button → paste it in the Chat ID field
3
Save & Test
Enter Token + Chat ID → click Test Connection → you should receive a test message in Telegram
4
Start Polling
Click Start Polling — the system will check every 5 seconds for your replies from Telegram
5
Reply to Tickets via Telegram
When a new ticket arrives, you get a Telegram message. Reply using:
/reply [ticketId] your message here
/close [ticketId] — close ticket
/pending [ticketId] — mark as pending
/list — list open tickets
Example: /reply 3 Thank you, we will fix this in 1 hour.
6
Integrate with Admin Panel
Both files share data via localStorage. Open admin panel and Telegram bot panel in the same browser — tickets stay in sync automatically.
